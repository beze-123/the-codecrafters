### Naming rules
Naming a variable in go is normally a letter like i,x,y,j,a or multiple words. 
Variable name cannot start with number.
It goes with underscore _.
It has no space
It is case-sensitive 

The sensitiveness of case depends on multiple words
         1. Camel case e.g first letter lower case,the first letter of the remaining words uppercase. 
         2. Pascal case e.g All the words start with capital letter
         3.Snake case e.g All words separated with underscore _.
